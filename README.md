KRAlert
===============

KRAlert is written in Swift wrapper of UIAlertController to simplify its usage in typical cases


## Installation
####CocoaPods

(Unfamiliar with [CocoaPods](http://cocoapods.org/) yet? It's a dependency management tool for iOS and Mac, check it out!)

Just add `pod 'KRAlert'` to your Podfile and run `pod install` in Terminal from your project folder.

Write `#import "KRAlert.h"` in files where you need to use KRAlert.
